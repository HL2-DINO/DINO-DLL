◆ GetSerializedToolDict()

void Holo2IRTracker::GetSerializedToolDict ( std::vector< double > &  out_encodedDoubleArray)

Populates the passed double vector as generated by IRTrackerUtils.

In V1.0 structure is as follows: [toolID, toolVisible(0/1), m00, m10, m20, ..., m33] repeats N times. Vector is of length 18 * N (N = number of tools).

Parameters
out_encodedDoubleArrayDouble array containing encoded info about tools.
