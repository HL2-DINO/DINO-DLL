◆ IntegerList

typedef std::vector<int> CorrespondenceMatcher::IntegerList

An index-matching list.

This type, represented as IntegerList, is designed to associate indices with points. It is typically expected to have the same length as your reference point list. To use it, create an IntegerList and populate it with indices that correspond to points in your reference point list. You can then access the indices to match points between sets.

Example usage:

IntegerList listPoints;
for (int i = 0; i < listPoints.size(); ++i)
{
std::cout << "Point " << i << " in the original set is Point " << listPoints[i] <<
" in the floating set.\n";
}
