void winrt::HL2DinoPlugin::implementation::HL2ResearchModeController::ToggleDisplaySensorImages ( bool  showTextures)

Toggle ability to send images for display on Unity side.

Pass in a bool to set whether the HL2 should actually process the sensor images so they can be displayed on the consumer/Unity side. If false, no extra operations done to copy/init display textures, so we can potentially run a little faster.
