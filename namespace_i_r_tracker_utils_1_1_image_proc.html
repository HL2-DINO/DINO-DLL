Enumerator
Basic 

Basic contour-detection filtering by thresholding, area and circularity using cv::findContours

Builds on the BlobDetectionMethod::Basic by enlarging the blob region to try a sub-pixel refinement of centre-estimation.

Helper function to increase dynamic range of image and then convert to an 8-bit image.

Internally scales each pixel by factor of 64 to 'brighten' and handles conversion from 16 to 8bit

Parameters
inputRaw16BitImgExpecting 16 bit AB image as obtained from HL2 AHAT depth sensor
output8BitImgA processed 8-bit AB image with an increased dynamic range

cv::convertTo will internally use OpenCV's saturateCast function. So linear remapping needed from above to avoid accidentally maxing out loads of pixels

Constructs a vector of valid InfraBlobInfo after cross-checking the depth values.

Iterates over the vector inblobPixels2D to check depth values. 2D pixel to 3D vector conversion done via the MapImagePointToUnitPlane function. outBlobInfo should then contain a list of valid 3D blobs which are likely to belong to infrared markers.

Parameters
inDepthImg16-bit depth image from HL2 without any processing required
inDepth2WorldTransform matrix of depth sensor to world coordinate system in this frame
inblobPixels2DDetected blob pixel locations (2D) to iterate over
MapImagePointToUnitPlanePointer to function that converts from 2D pixel locations (u,v) to the camera's unit plane (x,y,1)
outBlobInfoVector to populate with valid 3D blob info specified by InfraBlobInfo
